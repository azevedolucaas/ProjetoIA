# Projeto IA – Site Informativo sobre Inteligência Artificial

Este projeto é um **site estático em HTML e CSS** desenvolvido para apresentar conteúdos introdutórios sobre **Inteligência Artificial**, incluindo sessões como *Home*, *Sobre*, *Serviços* e *Autor*. O objetivo é demonstrar conhecimentos iniciais em desenvolvimento web com uma interface limpa e organizada.

---

## Tecnologias Utilizadas
- HTML5  
- CSS3  
- Git & GitHub Pages  

---

## Estrutura do Projeto

- index.hmtl
- style.css
- IMGs
